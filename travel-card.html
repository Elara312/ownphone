提示：使用CSS变量可以更方便地修改颜色和样式

💡 可用的CSS变量：

/* 主题色 */
--travel-card-bg-color: 背景色
--travel-card-border-color: 边框色
--travel-card-text-color: 文字色
--travel-card-time-color: 时间色
--travel-card-activity-color: 活动色
--travel-card-shadow-color: 阴影色
--travel-card-accent-color: 强调色
--travel-card-locked-color: 锁定色

/* 交互效果色 */
--travel-card-hover-bg: 按钮hover背景色
--travel-card-hover-overlay: 半透明hover效果

/* 背景设置 */
--travel-card-page-bg: 页面背景色
--travel-card-page-bg-image: 页面背景图片
--travel-card-container-bg: 容器背景色
--travel-card-card-bg-image: 卡片背景图片

/* 示例：修改为蓝色主题 */
:root {
    --travel-card-accent-color: #4A90E2;
    --travel-card-time-color: #5DADE2;
    --travel-card-shadow-color: #AED6F1;
    --travel-card-hover-bg: #3498DB;
}

/* 示例：设置背景图片 */
:root {
    --travel-card-page-bg-image: url(https://example.com/bg.jpg);
    --travel-card-card-bg-image: url(https://example.com/card-bg.jpg);
}
